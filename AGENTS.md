# Agents

This file is intended for AI agents and other automated tools that interact with the Pinekin codebase.

## Project Overview

Pinekin is a web-based game. The primary goal is to create an engaging and interactive experience for players.

## Development

The current technology stack is plain HTML, CSS, and JavaScript.

- **Game Logic**: Resides in `.js` files.
- **Structure**: Outlined in `index.html`.
- **Styling**: Handled by `.css` files.

When contributing, please adhere to the existing coding style and conventions.
